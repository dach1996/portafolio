Proyecto Administrador de Tickets - Movil

El proyecto presentado fué creado en la Universidad Israel (no implementado en la empresa), busca poder llevar registros de los eventos y tickets generados en el área de operaciones de la empresa kliente Strategik, este proyecto anteriormente fue realizado mediante el lenguaje Java para Desktop, ahora está implementado para una aplicación móvil.

Herramientas utilizadas:

  • C# - Xamarin
  • Mysql
  • API RESTful PHP
  • SignalR (Notificaciones en tiempo Real).
  • Almacenamiento de imagenes en el servidor.
  • Servicios nativos: Cámara, Huella digital.
  • Asincronismo (Mejorar rendimiento).
