Proyecto Administrador de Tickets.

El proyecto presentado fué creado para la empresa Kliente Strategik, busca solucionar el problema de llevar registros manuales de todos procesos y sucesos que ocurren en el área de operaciones. La aplicación cuenta con un login independiente por cada usuario lo cual permite determinar la persona que creó una tarea, también una ventana para poder escribir y dar seguimiento de cualquier novedad ocurrida durante el ticket.

Herramientas utilizadas:

  • Java swing
  • Mysql
  • Librerias de Notificación.
  • Implementación de Threads
